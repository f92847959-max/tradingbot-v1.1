    Create risk/kelly_calculator.py:
    - Class KellyCalculator with methods:
      - kelly_fraction(win_rate: float, avg_win: float, avg_loss: float) -> float
        Formula: f* = win_rate - (1 - win_rate) / (avg_win / avg_loss)
        Clamp result to [0.0, 0.25] (never risk more than 25%)
      - half_kelly(win_rate, avg_win, avg_loss) -> float: kelly_fraction() * 0.5
      - quarter_kelly(win_rate, avg_win, avg_loss) -> float: kelly_fraction() * 0.25
      - compute_from_trades(trades: list[dict]) -> float:
        Extract win_rate, avg_win_pips, avg_loss_pips from trade dicts with keys "net_pnl" and "pnl_pips"
        Require minimum 30 trades, return 0.0 if fewer
        Return half_kelly by default (conservative)
    - All methods are pure functions, no DB access, no async
    - Log every calculation with logging.getLogger(__name__)

    Create risk/volatility_sizer.py:
    - Class VolatilitySizer with __init__(baseline_atr: float = 3.0, min_scale: float = 0.25, max_scale: float = 1.5):
      - baseline_atr: "normal" ATR for XAUUSD (around 3.0 for 5min candles)
      - min_scale: minimum sizing factor (never go below 25% of base size)
      - max_scale: maximum sizing factor (never go above 150% of base size)
    - Methods:
      - calculate_atr_factor(atr: float) -> float:
        factor = baseline_atr / max(atr, 0.01)  # avoid div by zero
        return clamp(factor, min_scale, max_scale)
      - adjust_lot_size(base_lot: float, atr: float, min_lot_size: float = 0.01) -> float:
        factor = calculate_atr_factor(atr)
        return max(min_lot_size, round(base_lot * factor, 2))
    - Pure calculation, no DB, no async
    - Log factor and adjusted size at DEBUG level

    - Class AdvancedPositionSizer:
      __init__(
        base_risk_pct: float = 1.0,
        kelly_mode: str = "half",  # "full", "half", "quarter"
        baseline_atr: float = 3.0,
        min_lot_size: float = 0.01,
        max_lot_size: float = 10.0,
      )
      - Internally creates KellyCalculator and VolatilitySizer(baseline_atr=baseline_atr)
      - Stores _kelly_fraction: float = 0.0 (updated from trade history)
      - Stores _win_rate, _avg_win, _avg_loss for logging

    - set_trade_stats(win_rate: float, avg_win: float, avg_loss: float) -> None:
        Recalculates Kelly fraction based on kelly_mode
        Logs: "Kelly updated: mode={mode}, f*={fraction:.4f}, win_rate={win_rate:.2f}, RRR={avg_win/avg_loss:.2f}"

    - get_position_size(confidence: float, atr: float, account_balance: float) -> dict:
        1. If account_balance <= 0: return {"lot_size": 0.0, ...}
        2. Confidence tier: <0.6 = "low" (quarter_kelly), 0.6-0.8 = "medium" (half_kelly), >0.8 = "high" (full kelly)
           If _kelly_fraction == 0.0 (no trade history yet), use base_risk_pct as fallback
        3. risk_pct = kelly_adjusted_fraction * 100 (or base_risk_pct if no Kelly data)
        4. Apply volatility scaling: atr_factor = volatility_sizer.calculate_atr_factor(atr)
        5. base_lot = (account_balance * risk_pct / 100) / max(atr * 2, 0.01)  # SL at ~2 ATR
        6. adjusted_lot = volatility_sizer.adjust_lot_size(base_lot, atr, min_lot_size)
        7. Clamp to [min_lot_size, max_lot_size]
        8. Return dict: {
             "lot_size": float,
             "kelly_fraction": _kelly_fraction,
             "atr_factor": atr_factor,
             "confidence_tier": str,
             "risk_pct": risk_pct,
             "reasoning": f"Kelly {kelly_mode}={fraction:.4f}, ATR factor={atr_factor:.2f}, confidence={confidence_tier}"
           }
        9. Log at INFO: "Position size: {lot_size} lots (Kelly={fraction}, ATR_factor={atr_factor}, tier={tier})"

    - Module-level convenience functions (the Phase 10 interface):
      _instance: AdvancedPositionSizer | None = None

      def init_position_sizer(settings) -> AdvancedPositionSizer:
          global _instance
          _instance = AdvancedPositionSizer(
              base_risk_pct=settings.max_risk_per_trade_pct,
              kelly_mode=settings.kelly_mode,
              baseline_atr=settings.atr_baseline,
          )
          return _instance

      def get_position_size(confidence: float, atr: float, account_balance: float) -> float:
          # Convenience wrapper returning just the lot_size float
          if _instance is None:
              raise RuntimeError("AdvancedPositionSizer not initialized. Call init_position_sizer() first.")
          result = _instance.get_position_size(confidence, atr, account_balance)
          return result["lot_size"]

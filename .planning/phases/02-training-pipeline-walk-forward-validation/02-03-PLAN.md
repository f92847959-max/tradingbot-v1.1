## Task 1: Add `generate_training_report()` to `walk_forward.py`

Add a function to generate the walk-forward training report:

```python
def generate_training_report(
    windows_results: List[Dict[str, Any]],
    version_info: Dict[str, Any],
) -> Dict[str, Any]:
    """Generate walk-forward training report.

    Args:
        windows_results: Per-window results from WalkForwardValidator.run_all_windows()
        version_info: Version metadata (version string, training_date, etc.)

    Returns:
        Report dict suitable for JSON serialization + console output.
    """
```

The report structure:
```python
{
    "report_date": str,  # ISO format
    "version": str,
    "summary": {
        "n_windows": int,
        "total_test_samples": int,
        "window_type": "expanding",
    },
    "per_window": [
        {
            "window_id": int,
            "train_samples": int,
            "test_samples": int,
            "xgboost": {
                "accuracy": float, "f1": float,
                "win_rate": float, "profit_factor": float,
                "expectancy": float, "n_trades": int,
            },
            "lightgbm": { ... },
        },
    ],
    "aggregate": {
        "xgboost": {
            "win_rate": float, "profit_factor": float,
            "expectancy": float, "n_trades": int,
            "sharpe": float,
        },
        "lightgbm": { ... },
        "best_model": str,  # "xgboost" or "lightgbm" based on aggregate profit_factor
    },
}
```

**Critical: Aggregate computation.**
Per research pitfall 6: Do NOT average profit factors across windows. Instead:
- Collect total gross_profit and total gross_loss across all windows for each model
- Compute aggregate PF = total_gross_profit / total_gross_loss
- Compute aggregate win_rate = total_wins / total_trades
- Compute aggregate expectancy from aggregate win_rate and TP/SL
- Sum n_trades across windows

Also add a `print_training_report()` function that logs a formatted console summary:


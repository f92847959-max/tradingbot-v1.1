# Phase 12.7 Plan 01: Data Coverage Summary

Plan 12.7-01 adds early dataset coverage preflight and row-loss telemetry for AI training.

## Files

- `ai_engine/training/data_coverage.py` defines `DataCoverageError`, trainable-span diagnostics, row-loss reports, dataset manifests, and JSON writing.
- `ai_engine/training/data_source.py` now includes `duration_days` and `duration_hours` in dataframe summaries.
- `scripts/train_models.py` now supports `--source`, `--manifest-out`, and `--allow-short-data`, with DB/file/auto source preference before broker fallback when requested.
- `tests/test_training_data_coverage.py` covers span failure diagnostics, row-loss transitions, manifest keys, and JSON round-trip.
- `tests/test_training_data_source.py` covers new duration metadata.

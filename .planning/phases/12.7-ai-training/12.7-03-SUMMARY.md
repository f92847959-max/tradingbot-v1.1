# Phase 12.7 Plan 03: Promotion Gate Summary

Plan 12.7-03 adds a documented promotion gate and shadow-training manifest workflow.

## Files

- `ai_engine/training/promotion_gate.py` defines `PromotionGateConfig`, promotion evaluation, promotion-decision JSON writing, and shadow manifest creation.
- `ai_engine/training/pipeline.py` writes `promotion_decision.json` for every version and writes `shadow_training_manifest.json` only for approved decisions.
- `scripts/train_models.py` accepts `--champion-report` and enforces the promotion gate when a champion report is supplied.
- `tests/test_training_promotion_gate.py` covers pass/block decisions, calibration failure, drawdown failure, low non-HOLD support, artifact writing, and no production-pointer dependency for failed promotion.

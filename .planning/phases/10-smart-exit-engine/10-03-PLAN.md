# Phase 10 Plan 03: TP1 Partial Close

<objective>
Implement partial-close decision logic for Phase 10. When price reaches TP1, the exit engine should emit a one-time action to close 50% of the position and leave the remainder for trailing-stop management.
</objective>

<acceptance_criteria>
- BUY positions trigger at current_price >= tp1.
- SELL positions trigger at current_price <= tp1.
- Default close fraction is 0.5.
- No action is emitted before TP1.
- No duplicate action is emitted after TP1 was already handled for the deal.
- Unit tests cover BUY, SELL, not reached, duplicate prevention, and tracking reset.
</acceptance_criteria>

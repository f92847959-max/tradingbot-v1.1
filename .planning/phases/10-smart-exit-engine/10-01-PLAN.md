    ```python
    def calculate_dynamic_sl(
        direction: str,           # "BUY" or "SELL"
        entry_price: float,
        atr: float,               # Current ATR-14 value
        regime: MarketRegime,
        structure_levels: list[StructureLevel] | None = None,
        pip_size: float = 0.01,
        min_sl_pips: float = 5.0,
        structure_buffer_pips: float = 2.0,
    ) -> tuple[float, str]:
    ```

    Logic for `calculate_dynamic_sl`:
    1. Get `sl_atr_multiplier` from `get_regime_params(regime)` (TRENDING=1.5, RANGING=1.0, VOLATILE=2.0)
    2. Compute `atr_sl_distance = atr * sl_atr_multiplier`
    3. Compute `atr_sl`: for BUY = `entry_price - atr_sl_distance`, for SELL = `entry_price + atr_sl_distance`
    4. If `structure_levels` provided, find nearest relevant structure:
       - For BUY: find strongest support below entry (filter `level_type == "support"` and `price < entry_price`)
       - For SELL: find strongest resistance above entry (filter `level_type == "resistance"` and `price > entry_price`)
       - Add buffer: `structure_sl = structure_level.price - (structure_buffer_pips * pip_size)` for BUY, `+ buffer` for SELL
    5. Choose the more protective SL:
       - For BUY: `sl = max(atr_sl, structure_sl)` (higher is closer to entry = more protective)
       - For SELL: `sl = min(atr_sl, structure_sl)` (lower is closer to entry = more protective)
       - If no structure found, use atr_sl only
    6. Apply minimum floor: ensure SL distance >= `min_sl_pips * pip_size` from entry
    7. Return `(round(sl, 2), reason_string)` where reason is "atr", "structure", or "atr+structure"

    Also create helper `find_swing_levels(df: pd.DataFrame, lookback: int = 20, min_touches: int = 2) -> list[StructureLevel]`:
    - Scan last `lookback` candles for swing highs (local maxima in high column) and swing lows (local minima in low column)
    - A swing point is where `high[i] > high[i-1]` and `high[i] > high[i+1]` (vice versa for lows)
    - Group nearby levels within 0.5 ATR distance, count touches (occurrences in group)
    - Return list of StructureLevel with strength = number of touches
    - Filter: only return levels with `strength >= min_touches`

    `calculate_dynamic_tp(direction: str, entry_price: float, atr: float, regime: MarketRegime, candles: pd.DataFrame | None = None, pip_size: float = 0.01, min_tp_pips: float = 10.0) -> tuple[float, float | None, str]`:

    - Step 1: Get `tp_atr_multiplier` from `get_regime_params(regime)`

    - Step 3: If `candles` provided, call `find_sr_levels(candles)` to find structure

    - Return `(round(tp, 2), round(tp1, 2), reason)`

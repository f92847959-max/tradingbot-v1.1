**2. `parse_swarm_direction(report_markdown: str) -> tuple[str, float, str]` function:**
Parse German markdown report from MiroFish into (direction, confidence, reasoning_summary).
Use keyword matching with these German terms:

Bullish keywords: `aufwaertstrend, steigende preise, kaufsignal, bullish, preissteigerung, hausse, nachfrage steigt, positive entwicklung, zentralbank kauft, geopolitische unsicherheit steigt, inflationsdruck, dollar schwaecht`

Bearish keywords: `abwaertstrend, fallende preise, verkaufssignal, bearish, baisse, kurs faellt, dollar staerkt, zinsanstieg, restriktive geldpolitik, druck auf goldpreis, risikoappetit steigt`

Logic: count keyword matches in lowercased text. If bullish_count > bearish_count -> "BUY". If bearish_count > bullish_count -> "SELL". Otherwise -> "NEUTRAL". Confidence = min(0.9, 0.5 + margin / (total * 2.0)). Empty text -> ("NEUTRAL", 0.5, "Keine klare Richtung erkennbar").

Add to `MiroFishClient` class:

```python
def check_veto(self, signal: dict) -> dict:
    """Apply MiroFish veto check to an ML signal (D-06 to D-09).

    - If no cached assessment: return signal unchanged (D-16)
    - If swarm agrees or is NEUTRAL: return signal with mirofish metadata
    - If swarm contradicts (BUY vs SELL): convert to HOLD with veto reason (D-09)
    """
    assessment = self.get_cached_assessment()
    if assessment is None:
        return signal  # No cache -- trade without MiroFish (D-16)

    ml_action = signal.get("action", "HOLD")
    swarm_direction = assessment.direction

    # Veto: swarm directly contradicts ML signal
    if (
        (ml_action == "BUY" and swarm_direction == "SELL") or
        (ml_action == "SELL" and swarm_direction == "BUY")
    ):
        veto_reason = (
            f"MiroFish Veto: Schwarm={swarm_direction}, ML={ml_action}. "
            f"{assessment.reasoning}"
        )
        logger.info("Trade BLOCKED by MiroFish: %s", veto_reason)
        return {
            **signal,
            "action": "HOLD",
            "mirofish_veto": True,
            "mirofish_reasoning": veto_reason,
        }

    # Agreement or neutral -- trade proceeds
    return {
        **signal,
        "mirofish_veto": False,
        "mirofish_direction": swarm_direction,
        "mirofish_confidence": assessment.confidence,
        "mirofish_reasoning": assessment.reasoning,
    }
```
